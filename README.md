# *PALAMEDES* is a multi games plataform made with PYTHON and GO

### FEATURING

* Pong (2 versions with PYGAME)
* Circle-Pong
* Breakout (PYGAME and GO)
* Shogu
* Peg-solitarie
* **_and more games to come!_** :space_invader:
